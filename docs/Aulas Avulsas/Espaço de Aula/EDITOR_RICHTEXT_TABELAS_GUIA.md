# 📊 Guia de Tabelas no Editor RichText

## 🎯 Como Usar Tabelas no Bloco de Anotações

### ➕ **Inserir uma Nova Tabela**

1. Clique no botão **📊 Tabela** na barra de ferramentas
2. Uma tabela 3x3 com cabeçalho será inserida automaticamente
3. A tabela terá:
     - **3 colunas** e **3 linhas**
     - **Primeira linha como cabeçalho** (com destaque visual)
     - **Bordas e estilos** aplicados automaticamente

### ✏️ **Editar Conteúdo da Tabela**

1. **Clique em qualquer célula** para começar a digitar
2. Use **Tab** para navegar para a próxima célula
3. Use **Shift + Tab** para voltar à célula anterior
4. **Enter** cria uma nova linha dentro da célula
5. **Setas do teclado** também permitem navegar entre células

### 🔧 **Controles Avançados de Tabela**

Quando o cursor estiver **dentro de uma tabela**, você verá:

#### 🏷️ **Indicador Visual**

-    **Chip "📊 Editando Tabela"** aparece para mostrar que você está no modo tabela
-    Controles organizados em seções claras com labels

#### 📋 **Interface Organizada**

Os controles aparecem organizados em **3 seções bem definidas**:

#### 📏 **1. Seção "Linhas:"**

-    **Label clara**: "Linhas:" aparece antes dos botões
-    **➕ Ícone normal**: Adiciona linha acima
-    **➕ Ícone invertido**: Adiciona linha abaixo
-    **🗑️ Ícone vermelho**: Remove linha atual

#### 📐 **2. Seção "Colunas:"**

-    **Label clara**: "Colunas:" aparece antes dos botões
-    **➕ Ícone normal**: Adiciona coluna à esquerda
-    **➕ Ícone invertido**: Adiciona coluna à direita
-    **🗑️ Ícone vermelho**: Remove coluna atual

#### ❌ **3. Botão "Deletar Tabela"**

-    **Botão vermelho com texto**: "Deletar Tabela"
-    **Ícone + texto juntos** para máxima clareza
-    ⚠️ **Ação definitiva**: Remove toda a estrutura da tabela

### 🎨 **Recursos Visuais**

#### ✨ **Feedback Visual**

-    **Células destacadas** ao passar o mouse
-    **Bordas coloridas** na tabela selecionada
-    **Cabeçalho diferenciado** com fundo cinza
-    **Células mínimas** de 100px de largura

#### 🎯 **Seleção e Foco**

-    **Célula ativa** fica destacada
-    **Tabela selecionada** mostra borda azul
-    **Controles contextuais** aparecem automaticamente

### ⌨️ **Atalhos de Teclado**

| Ação                 | Atalho                      |
| -------------------- | --------------------------- |
| Próxima célula       | `Tab`                       |
| Célula anterior      | `Shift + Tab`               |
| Nova linha na célula | `Enter`                     |
| Sair da tabela       | `Esc` (e navegar com setas) |
| Desfazer             | `Ctrl + Z`                  |
| Refazer              | `Ctrl + Y`                  |

### ⚠️ **Como Deletar uma Tabela**

#### **Método 1: Botão de Deletar (Recomendado)**

1. Clique em qualquer célula da tabela
2. Os controles de tabela aparecerão na toolbar
3. Clique no botão **🗑️ Deletar Tabela** (último grupo)
4. A tabela será removida instantaneamente

#### **Método 2: Seleção Manual**

1. Posicione o cursor **antes** da tabela
2. Arraste para selecionar a tabela inteira
3. Pressione `Delete` ou `Backspace`

#### **Método 3: Desfazer**

1. Se acabou de criar a tabela, use `Ctrl + Z`
2. Isso desfaz a ação de inserção

### 💡 **Dicas de Uso**

#### ✅ **Boas Práticas**

-    **Use cabeçalhos** para organizar informações
-    **Mantenha células** com conteúdo conciso
-    **Combine com formatação** (negrito, itálico) nas células
-    **Teste a navegação** com Tab para fluidez

#### 🚫 **Evite**

-    **Tabelas muito largas** em dispositivos móveis
-    **Muitas colunas** (máximo 5-6 recomendado)
-    **Células vazias** desnecessárias
-    **Conteúdo muito longo** em uma célula

### 📱 **Responsividade**

#### **Desktop**

-    Tabelas ficam com largura total disponível
-    Todas as funcionalidades ativas
-    Controles completos na toolbar

#### **Mobile/Tablet**

-    Tabelas podem fazer scroll horizontal
-    Controles simplificados
-    Célula mínima mantida para legibilidade

### 🔄 **Integração com Outras Funcionalidades**

#### **Auto-Save**

-    ✅ **Mudanças salvas automaticamente** ao editar células
-    ✅ **Estrutura preservada** entre sessões
-    ✅ **Sincronização** em tempo real

#### **Formatação**

-    ✅ **Negrito/Itálico** funciona dentro das células
-    ✅ **Links** podem ser adicionados nas células
-    ✅ **Código inline** funciona nas células
-    ❌ **Listas** não recomendadas dentro de células

### 🛠️ **Solução de Problemas**

#### **Tabela não aparece**

-    Verifique se está no modo de edição (não read-only)
-    Certifique-se que há espaço suficiente no editor

#### **Controles não aparecem**

-    Clique dentro de uma célula da tabela
-    Os controles aparecerão automaticamente na toolbar

#### **Não consigo deletar**

-    Use o botão específico "Deletar Tabela" quando estiver dentro da tabela
-    Ou posicione o cursor fora da tabela e selecione-a manualmente

#### **Formatação perdida**

-    Tabelas mantêm formatação no formato HTML
-    Se convertidas para texto, a estrutura pode ser perdida
